# 🎓 Active Student Form (Task Management App)

## 📌 Project Description
This is a simple Active Student Form application where students can be added and deleted dynamically. The project works like a basic task management system, where each student entry acts as a task.

This project is built using **HTML, CSS, and JavaScript** to practice core frontend development concepts like form handling, DOM manipulation, and dynamic UI updates.

---

## 🚀 Features
- ➕ Add new student details
- ❌ Delete student from the list
- 📋 Display all active students in a list
- ⚡ Real-time updates when adding or deleting students
- 🎯 Simple and clean user interface

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 📸 How It Works
1. User enters student details in the input form  
2. Clicks the **Add** button  
3. Student gets added to the list  
4. Each student appears like a task item  
5. Click **Delete** button to remove the student from the list
